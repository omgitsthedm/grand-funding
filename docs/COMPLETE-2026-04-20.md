## If something breaks

- **Premium QA gate fails on push** → run `npm run test:premium` locally to see exact failure before pushing again
- **Form submissions missing fields** → Netlify form-detection may need redeploy; check Site Settings → Forms
- **Mobile video not playing** → confirm `<source media>` selection in devtools; on iOS, `preload="auto"` is required
- **Lighthouse mobile perf drops** → check Search Console for Core Web Vitals field data before making code changes; simulator variance is ±15 points
